# 📊 Stock Tracker - Indian Market

A simple React-based stock tracker web app for Indian stock market simulation.

---

## 🚀 Features

- Live stock price simulation
- Add / Remove Watchlist
- Search stocks
- Dark mode
- Stock chart visualization
- Popup stock details

---

## 🛠️ Tech Stack

- React JS
- JavaScript
- HTML
- CSS
- Chart.js
- Vite
